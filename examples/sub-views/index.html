Sub Views

Sub-views are views that can be set with other views. This is useful to create reusable UI structures.

Each subview is registered on the view typically during the initialization phase. These views are registered using the addView function. The can be registered with additional information like:

  • selector: a jquery selector which would identify the DOM element to attach the view
  • alias: an optional alias which can be used for event bubbling

All sub-views will be properly destroyed when the parent destroy method is called. The will also automatically receive a show and hide method (if they do not already have one). 

...
initialize: function() {
	this.$uper('initialize', arguments);
	
	var myView = this.addView({
		view: new SimpleView(),
		alias: 'myView',
		selector: '#myView'
	});
}
