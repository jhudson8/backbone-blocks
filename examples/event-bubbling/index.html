Event Bubbling

Sub-views and collections can have their events automatically bubbled up to the parent view. Parent views can also use the events hash to bind to both sub-views and collection events as well as parent view events. The alias value is used as a prefix for all events. 

...
events: {
	myView: {
		rendered: 'onSubViewRendered',
		thing1: 'onThing1'
	}
	// OR
	'myView:rendered': 'onSubViewRendered',
	'myView:thing1': 'onThing1'
},

initialize: function() {
	this.$uper('initialize', arguments);
	
	var myView = this.addView({
		view: new SimpleView(),
		alias: 'myView',
		selector: '#myView'
	});
}
